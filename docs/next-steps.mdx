## Project Commands

```bash
npm run start    # start Expo dev server
npm run ios      # run iOS target
npm run android  # run Android target
npm run web      # run web target
```

## Checklist Before Demo/Submission

1. Confirm `.env` values and API key are loaded correctly.
2. Verify both tabs work: `Home` and `7-Day`.
3. Check that known weather codes do not show fallback `?` icons.

## Short Roadmap

- Add city search input (city input + submit).
- Add unit toggle (`°C` / `°F`).
- Move to a shared weather state source to avoid duplicate fetches.
- Add basic tests for helper functions.

## Documentation (docs.page)

1. Edit pages in the `docs/` folder.
2. Preview locally via `https://docs.page/preview`.
3. Push to GitHub — the public docs update automatically.
