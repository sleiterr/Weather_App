This documentation covers the Weather App built with React Native and Expo Router.
The project demonstrates WeatherAPI integration, custom UI, an hourly forecast, and a 7-day forecast screen.

## Quick Start

### Requirements

- Node.js 18+
- npm

### Run Locally

```bash
npm install
npm run start
```

### Additional Commands

```bash
npm run ios
npm run android
npm run web
```

## Tech Stack

- Expo
- React Native
- Expo Router
- WeatherAPI
- Ionicons
