## Folders and Responsibilities

- `app/` — screens and routing with Expo Router.
- `components/features/` — larger feature blocks (grid, hourly row, nav content).
- `components/ui/` — reusable UI elements.
- `components/layout/` — layout/navigation parts (including custom TabBar).
- `hooks/` — API fetching and loading state logic.
- `constants/` — color theme and API configuration.
- `util/` — formatting helpers and icon mapping.
- `docs/` — docs.page documentation.

## Important Files

- `hooks/useWeatherApi.js` — main weather data fetch logic.
- `app/(tabs)/index.js` — Home screen.
- `app/(tabs)/favorites.js` — 7-Day Forecast screen.
- `components/features/HourlyForecastRow.js` — hourly forecast row.
- `util/weatherHelpers.js` — weather-code-to-icon mapping.
